# Solty - Readme

tysm for powercord community for help <3
## Installation

Use the Following Command in your powercord directory to install the theme:

```bash
git clone https://github.com/trujesz/soltycord
```

## Featuring:
* Dracula theme mixed
* Fonts
* Color change
* Scrollbar for long custom statuses
